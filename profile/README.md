## Why 3DMark Time Spy?

Unlike basic stress tests, **3DMark Time Spy** offers dual rendering paths, physics engine simulation, and clock behavior monitoring. It integrates seamlessly with your existing hardware diagnostic workflow (temperature loggers, overlay tools, monitoring software) and provides real-time score comparison across all devices. Whether you validate overclocks or test new graphics cards, **3DMark Time Spy** adapts to your performance profile.

**Key benefits:**
- Hardware-first — all benchmark logs stay local with optional cloud sync  
- Cross-platform — use on desktops, laptops, and external GPU enclosures  
- Offline mode — benchmarking works without internet, syncs results when connection returns

## Why This Template Works

This structure solves the "empty benchmark slate" problem. New users often struggle with where to start configuring GPU tests. This proven layout provides: **Areas** for hardware testing (Overclocking, Stock, Undervolt, Thermal), **Projects** for evaluation tasks (Stability validation, Score optimization), and **Tags** for risk level (@high, @low) and component type (@gpu, @cpu). By adopting this system, you gain pre-built perspectives like "Critical Errors" (crashes detected today) and "Monthly Review" (cards with declining scores).
